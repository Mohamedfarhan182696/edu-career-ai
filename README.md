# edu-career-ai
AI-Powered Automated System for Faculty Career Advancement in Higher Education  An intelligent platform that helps faculty members track their performance, get personalized career recommendations, predict promotion chances, and plan their professional growth. Built with FastAPI, React, XGBoost, and Sentence Transformers.

# EduCareerAI - AI Automated Faculty Career Advancement System

**An Intelligent Platform for Higher Education Institutions (NAAC, NBA, NIRF Ready)**

## Features
- AI-powered CV, Publication & Profile Analysis
- Personalized Career Roadmap Generator
- Promotion Probability Prediction (ML Model)
- Opportunity Recommendation (Grants, Journals, Conferences)
- Performance Analytics Dashboard
- Automated Progress Reports & Alerts

## Tech Stack
- **Backend**: FastAPI + PostgreSQL
- **AI/ML**: LangChain, Sentence-Transformers, XGBoost
- **Frontend**: React + TypeScript + Tailwind
- **Deployment**: Docker
